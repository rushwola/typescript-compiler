# 项目目录结构

1. include 目录用于存放项目的头文件
2. src 目录用于存放项目的源代码
3. bin 目录用于存放项目生成的可执行文件
4. lib 目录用于存放生成的库文件
5. doc 目录用于存放项目的文档
6. test 目录用于存放测试代码
7. build 目录用于存放构建生成的中间文件
8. tools 目录用于存放编译所需的工具
9. resource 目录用于存放项目所需的资源文件
